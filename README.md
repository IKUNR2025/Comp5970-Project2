# Comp5970 Project 2 - 2D Platformer

This is a 2D platformer game made in Unity for COMP5970 Project 2.

The game's name is Coin Adventure

## Features

- Complete playable platformer level
- Player movement, jumping, and double jumping
- Start area and goal area
- Collectible coins
- Moving enemy hazard
- Win and lose conditions
- Start menu and restart button
- Custom 2D visual style

## Controls

- A / D or Left / Right Arrow: Move
- Space: Jump
- Space again in air: Double jump

## How to Play

Click **Start Game** to begin. Collect all coins, avoid the enemy, and reach the goal area to win. The goal only works after all coins are collected. If the player touches the enemy, the game is over.

## Unity Version

Unity 6.3 LTS  
Editor Version: 6000.3.16f1
